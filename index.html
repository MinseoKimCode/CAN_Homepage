About Us

CAN이란?

Computer Algorithm Network — 알고리즘과 개발, 그리고 네트워크로 함께 성장합니다.

우리의 이야기

CAN은 1988년 학부 최초로 컴퓨터공학과 학생들이 모여 설립한 학술 동아리입니다. 이론 공부에 그치지 않고 실제 프로젝트와 알고리즘 스터디를 통해 실력을 키워나갑니다.

정기적으로 프로젝트 팀 활동 등 다양한 프로그램을 운영하며 선후배 간의 멘토링 문화를 이어가고 있습니다.

학습

알고리즘, 웹/앱 개발, AI 등 다양한 분야를 함께 공부합니다.

협업

팀 프로젝트를 통해 협업 능력과 실무 경험을 쌓습니다.

성장

멘토링과 코드 리뷰 문화로 서로를 성장시킵니다.

can.cpp 
#include <format>
#include <iostream>
#include <vector>

struct CAN {
  std::string name    = "CAN";
  int founded = 1988;
  int members = 80;

  auto mission() const
      -> std::string {
    return "CAN에서 같이 성장할 사람~~~~????";
  }

  auto values() const
      -> std::vector<std::string> {
    return {"빨리", "지원하기", "누르자"};
  }
};

int main() {
  const CAN can{};
  std::cout << std::format(
    "[{}] {}\n",
    can.name, can.mission()
  );
}
